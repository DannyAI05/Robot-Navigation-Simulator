# 🤖 Robot Control Simulator

A Java Swing application that simulates an intelligent robot navigating a 10×10 grid while avoiding obstacles and reaching a target.

## Features

- Interactive Java Swing GUI
- Robot movement controls
- Keyboard shortcuts (W, A, D, R)
- Battery management system
- Obstacle avoidance
- BFS pathfinding
- Mission completion detection
- Status dashboard
- Clean object-oriented design

## Technologies

- Java
- Swing
- OOP
- Breadth-First Search (BFS)

## Future Improvements

- A* pathfinding
- Animated robot sprites
- Sound effects
- Multiple maps
- Save/load game
- Random obstacle generation
