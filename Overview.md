# 🤖 Robot Control Simulator

## Project Overview

The **Robot Control Simulator** is a desktop application developed in Java Swing that simulates the operation of an autonomous mobile robot navigating a two-dimensional environment. The simulator allows users to control a robot manually while monitoring its battery level, movement, orientation, mission progress, and interaction with obstacles.

The project demonstrates object-oriented software engineering principles by modeling real-world robotic behavior through classes, graphical user interfaces, and intelligent navigation algorithms. It provides an interactive environment for understanding how robots perceive, navigate, and accomplish tasks within constrained environments.

---

# Problem Statement

Many robotics concepts are difficult to understand without physical hardware. Purchasing and programming robots can be expensive, making practical learning inaccessible to many students.

This project addresses that challenge by providing a software-based robot simulator that enables users to experiment with robot navigation, obstacle avoidance, and mission execution without requiring any physical robotic equipment.

---

# Solution

The simulator recreates a robot operating inside a virtual 10×10 environment where it must:

* Navigate safely across the map
* Avoid obstacles
* Manage limited battery power
* Reach a designated target
* Display its current operational status in real time

Users can control the robot manually while observing how its internal state changes during execution.
